# SpringShopMVC
My Spring MVC pattern Shop project.

1. Shop 프로젝트 진행 목적
- 팀프로젝트에서 수행하지 못했던 장바구니, 파일업로드, 상품등록 등을 개인 프로젝트에서 연습하고 googleChart API를 이용하여 간단한 백엔드 차트분석을 경험하기 위함.
